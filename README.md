# Employee Payroll System (Java)

## Description
A simple Java console application to manage employee payroll. Users can add employees, automatically calculate bonus (10%), tax (15%), and net salary, and view all payroll details. Demonstrates OOP concepts like classes, methods, and ArrayList in a menu-driven system.

---

## Features
- Menu-driven interface:
  1. Add Employee
  2. View Payroll
  3. Exit
- Automatic payroll calculations
- Supports multiple employees using `ArrayList`
- Displays employee info neatly using `toString()`

---

## Technologies Used
- Java 8 or above
- Object-Oriented Programming (OOP)
- Collections (`ArrayList`)
- Console input/output (`Scanner`)
